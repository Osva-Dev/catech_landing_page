## CATECH Landing Page Documentation

## Que es CATECH?

CATECH es una startup de servicios digitales especializada en la creación de
landing pages estratégicas, infraestructura digital y soluciones web orientadas a
conversión para emprendedores, negocios locales y pymes en México.

## Estructura

Buscamos implementar una pagina web para atraer clientes en base a paquetes, cada paquete tiene diferentes caracteristicas y precios y lograr realizar su respectiva pagina web.
